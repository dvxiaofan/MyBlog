# MyBlog

## 个人博客

基于 VuePress v2 搭建的个人技术博客

## 访问地址

https://dvxiaofan.github.io/

## 技术栈

- VuePress v2.0.0-rc.26
- Vue 3.4.0
- Vite 7.1.12

## 开发

需要 Node.js 20+ 版本（推荐使用 nvm 管理）

```bash
# 启动开发服务器
nvm exec 22.18.0 npm run dev

# 构建生产版本
nvm exec 22.18.0 npm run build

# 部署到 GitHub Pages
nvm exec 22.18.0 npm run deploy
```
